# 📦 Project Name: PySpark Sentiment Analysis on Amazon Fine Food Reviews

A complete end-to-end sentiment analysis pipeline built using PySpark and Google Colab, analyzing Amazon Fine Food Reviews.

---

## 📁 Project Structure
```

```

---

## ✅ Steps Performed

1. **Data Loading & Cleaning**
2. **Sentiment Labeling** → (Score ≥ 4 = Positive, ≤ 2 = Negative)
3. **Text Preprocessing** → Tokenization, StopWords Removal, TF-IDF
4. **Model Training** → Logistic Regression with PySpark MLlib
5. **Model Evaluation** → Accuracy & F1 Score
6. **Model Saving & Export**
7. **Prediction CSV Export**

---

## ⚙️ Method Used
- **Binary Sentiment Classification** using review scores
- **TF-IDF Feature Extraction** for vectorizing text
- **Logistic Regression** for supervised classification
- **PySpark ML Pipelines** to streamline data and model operations

---

## 📊 Results
- **Accuracy**: `86%`
- **F1 Score**: `0.86`
